# Модель оцінки ефективності реклами в соціальних мережах (CERI Dashboard)

Цей репозиторій містить програмну реалізацію моделі та інформаційної технології для оцінювання ефективності рекламних кампаній у соціальних мережах. 

Проєкт розроблено в рамках дисертаційного дослідження: **«Моделі системи та інформаційної технології підвищення ефективності реклами в соціальних мережах»** (Автор: Роїк О. О., Національний університет «Львівська політехніка», 2026).

## 📌 Опис проєкту

Система дозволяє агрегувати дані з соціальних сторінок, розраховувати базові та комплексні метрики залученості, а також візуалізувати їх для прийняття ефективних маркетингових рішень. 

Головною особливістю системи є використання авторського показника **CERI (Comprehensive Engagement and Revenue Index)**, який об'єднує фінансові показники (ROI, CPE) та метрики залученості (WES, FCF) для комплексної оцінки кожного окремого допису.

## 🚀 Основні можливості

* **Інтеграція з соціальними мережами:** Збір даних про публікації, охоплення, лайки, коментарі та поширення.
* **Глибока аналітика (Backend):** Автоматичний розрахунок показників:
    * `ER` (Engagement Rate)
    * `ERR` (Engagement Rate by Reach)
    * `ROI` (Return on Investment)
    * `CPE` (Cost Per Engagement)
    * `WES` (Weighted Engagement Score)
    * `FCF` (Follower Conversion Factor)
    * `CERI` (Комплексний індекс залученості та доходу)
* **Інтерактивний дашборд (Frontend):** Динамічна таблиця з можливістю налаштування вагових коефіцієнтів ($W_1$, $W_2$), витрат та доходів у реальному часі, а також сортування публікацій за ефективністю.

## 🛠 Технологічний стек

**Backend:**
* Python 3.x
* Django & Django REST Framework (DRF)
* SQLite / PostgreSQL (для зберігання даних)

**Frontend:**
* React.js
* Axios (для API запитів)
